# AGENTS

## Workflow

- Prerequisite: Node.js 20 or newer.
- Install dependencies with `npm install`.
- Start the dev server with `npm run dev`.
- The checked-in Vite log shows local development has been run at `http://127.0.0.1:4173/`.
- Build production assets with `npm run build`.
- Preview the production build with `npm run preview`.

## Notes

- This is a client-only React app built with Vite.
- `package.json` only defines `dev`, `build`, and `preview`; there is no configured test or lint command yet.
